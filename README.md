Exercícios de Lógica em Python

Este repositório contém exercícios básicos de lógica de programação desenvolvidos em Python.

📂 Exercícios

 ° calculo_media.py
  Programa que calcula a média de notas informadas pelo usuário.

 °classificacao_idade.py
  Classifica a pessoa em categorias (criança, adolescente, adulto, idoso).

 °impar_par.py
  Verifica se um número é par ou ímpar.

 °verificar_idade.py
  Informa se a pessoa é maior ou menor de idade.

  °calculadora.py
  Programa que realiza operações matemáticas básicas (soma, subtração, multiplicação e divisão).

 Objetivo:
 Praticar conceitos básicos de:

* Entrada de dados (`input`)
* Estruturas condicionais (`if/else`)
* Operações matemáticas
